# Kitchen Scale User Manual

## Key Features
1. Maximum capacity options: 1 kg, 2 kg, or 3 kg models (this unit: up to 3000 g).
2. High precision: 0.1 g resolution.
3. Three units: g, oz, and ml.
4. Touch controls with audible key feedback.
5. Built-in timer (count-up and countdown).
6. USB rechargeable battery with battery level indicator.
7. Tare (zero) function for container weighing.

## 1. Specifications
1. Unit options: `g`, `oz`, `ml`.
2. Main key: `POWER/TIMER` (touch).
3. Auto power-off: 60 seconds (when idle).
4. Manual power on/off: touch and hold `POWER/TIMER` for 2 seconds.
5. Minimum measurable weight: 1 g.
6. Maximum capacity: 3000 g.
7. Overload threshold: above rated limit (display shows overload warning).
8. Scale resolution: 0.1 g, 0.005 oz, 0.1 ml.
9. Power-on self-test display:

```
+--------------------------+
| 88:88  88.8.8.8 oz ml g  |
+--------------------------+
```

10. Power supply: 3.7 V rechargeable lithium battery, 400 mAh.
11. Audible prompt: one short beep when keys are pressed.
12. Tare/Zero: in weighing mode, press `ZERO` to reset displayed weight to zero.

## 2. Charging and Display Indicators
1. USB charging: when the USB cable is connected, the display shows battery level and a charging icon.
2. Full charge: battery icon shows all bars. Typical full charge time is about 2 to 5 hours.
3. While charging: weighing can be used; some timer operations may be limited.
4. Battery display fallback: after 60 seconds of no timer operation, the display may return to battery status.
5. Overload warning display:

```
+------------------+
| 0:00   EEEEE     |
+------------------+
```

6. Low-voltage warning (below 3.0 V):

```
+------------------+
| 0:00   Lo        |
+------------------+
```

7. Unstable-weight warning:

```
+------------------+
| 88:88  UnSt      |
+------------------+
```

## 3. Unit Conversion
1. Default unit at startup is `g`.
2. Touch and hold `POWER/TIMER` for 2 seconds to enter unit switching.
3. Each long press cycles units in this order: `g -> oz -> ml -> g`.
4. Unit conversion flow:

```
1. Power On
   └──▶ Press Unit Key
       ├───▶ Display in Grams
       ├───▶ Display in Ounces
       └───▶ Display in Milliliters
```

## 4. Timer Functions
1. Count-up timer: press `POWER/TIMER` once to start.
2. Pause/resume: short press `POWER/TIMER` again.
3. Exit count-up mode: double-press `POWER/TIMER` to clear and exit.
4. Countdown setup: triple-press `POWER/TIMER`.
5. In countdown setup:
   - press `POWER/TIMER` to switch between minutes and seconds.
   - press `ZERO` to adjust/set values.
6. Maximum timer value: 99 minutes 59 seconds.
7. Countdown alarm: when time reaches zero, the scale beeps for up to 1 minute; press any key to stop.

## 5. Basic Weighing Operation
1. Place the scale on a stable, flat, hard surface.
2. Touch and hold `POWER/TIMER` for 2 seconds to power on.
3. Wait for startup sequence:

<pre>
+------------------------------------+      +------------------------------------+
|      88:88  88.8.8.8 oz ml g       |  ->  |            0:00   0.0g             |
+------------------------------------+      +------------------------------------+
</pre>

4. Place your container or ingredient on the platform.
5. Press `ZERO` to tare if needed, then add ingredients.

<pre>
+------------------------------------+      +------------------------------------+
| 0:00  200.0g        oz ml g  ZERO  |  ->  | 0:00   0.0g         oz ml g  ZERO  |
+------------------------------------+      +------------------------------------+
</pre>

## 6. Safety and Care
1. Do not disassemble or modify the scale. Unauthorized changes void the warranty.
2. Use only the specified rechargeable battery type and charging method.
3. Incorrect battery handling may cause overheating, leakage, or explosion.
4. Keep the scale dry and clean. Wipe with a soft, slightly damp cloth.
5. Do not overload the scale beyond its rated capacity.
